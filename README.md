# AI 经验库

个人 AI 协作经验沉淀库，跨项目长期积累。

## 目录结构

```
docs/AI经验库/
├── 04_复用启动词.md          ← 全局启动词模板
├── 待整理/                    ← 临时存放，不分项目
├── 项目经验/                   ← 按项目维度组织
│   ├── OmniFlow/
│   ├── EC-Copilot/
│   └── 三科/
└── 通用Skill/                  ← 跨项目可复用
```

## 使用方式

1. 对话中有价值内容 → 扔进 `待整理/`
2. 定期让 AI 判断 → 沉淀为项目经验卡片或 Skill 卡片
3. 启动新 AI 对话 → 从 `04_复用启动词.md` 复制对应模板
