## 核心流程
1. 产品透明图 → 产品资产库
2. 主图模板 → JSON 管理
3. 表格导入 → SKU、价格、卖点、活动标签
4. SKU 匹配 → 产品透明图
5. 一行表格 → 一张主图
6. 批量预览 → 检查异常
7. 批量导出 → PNG / JPG / ZIP
8. 后续 → PSD 导入生成模板 JSON

## 技术栈
- 原生 HTML / CSS / JavaScript
- Canvas 渲染与导出 PNG
- 本地 FileReader 读取产品图、模板 PNG 层、模板配置 JSON
- CSV 文本解析后映射为结构化 `brief`

## 已确认能力
- 3 个主图线框模板：左机制右产品、上机制中产品、双11预售固定分层。
- 5 个活动背景预设。
- 上传 PS 导出的背景层、右侧底板、价格条底板 PNG。
- 导出/导入活动模板配置 JSON。
- 单张产品图上传、批量产品素材按文件名匹配。
- 产品图缩放、X/Y 偏移。
- CSV / Excel 字段驱动批量套版。
- 右侧内容槽自动切换：有赠品机制时显示赠品，没有赠品时显示卖点。
- 导出前检查缺字段、缺图片、模板 ID、文案长度。
- 按 `商品ID-命名号-产品名称.png` 导出 PNG。
