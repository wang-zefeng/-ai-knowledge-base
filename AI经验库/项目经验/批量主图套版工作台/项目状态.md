# 批量主图套版工作台 - 项目状态

## 定位
基于「模板库 + 产品资产库 + 表格数据」的批量主图生成系统

## 核心流程
1. 产品透明图 → 产品资产库
2. 主图模板 → JSON 管理
3. 表格导入 → SKU、价格、卖点、活动标签
4. SKU 匹配 → 产品透明图
5. 一行表格 → 一张主图
6. 批量预览 → 检查异常
7. 批量导出 → PNG / JPG / ZIP
8. 后续 → PSD 导入生成模板 JSON
