## Skill：千川视频自动剪辑

### 用途
用产品素材 + Excel 脚本批量生成千川竖屏视频，自动匹配画面、生成配音、字幕、花字和配乐。

### 适用场景
需要批量产出千川投放素材时，替代手动剪辑。

### 输入
- 素材目录：产品实拍、内料质地、模特效果、背景音乐
- Excel 脚本：动作、画面、语言/口播/文案字段
- TTS 密钥（火山/豆包），存入本地 `secrets.local.json`，不写入经验库

### 执行步骤
1. 准备素材目录和 Excel 脚本
2. 配置 TTS 密钥到本地 `secrets.local.json`
3. 调用 `qianchuan-video` Skill
4. 批量生成视频：1080x1920、30fps、AAC 音轨
5. 配音按脚本文案逐句生成，字幕和画面跟随配音时间轴
6. 背景音乐按素材池轮换，随机起点
7. 用 `ffprobe` 检查分辨率、帧率、音轨和时长

### 输出结果
批量竖屏视频，配音字幕画面逐句同步，时长不超过限制值。

### 可复用提示词
```
使用 `qianchuan-video` Skill，根据这个素材文件夹和脚本文件夹，批量生成 10 条千川竖屏视频。要求：配音、字幕和画面逐句同步；视频时长跟随配音但不超过 35 秒；每条使用不同背景音乐；去除原声；字幕不要出现反斜杠；禁止静止补帧，画面要流畅。
```

### 注意事项
- 强行压成 15 秒会导致配音过快
- 用 `tpad=stop_mode=clone` 会造成画面假卡顿，应循环动态素材或拆镜头
- ASS 字幕里的 `\N` 如果二次转义，会在画面显示反斜杠
- 停止批处理时要检查 Python/FFmpeg 子进程是否仍在跑
- TTS API Key 不能写进 MD、模板、日志或 Skill 包
- 长脚本不能硬塞，应按语义截取或限制最大时长
- 换设备复用时需重新安装 Skill、Python、FFmpeg，并配置本地 TTS Secrets

给出素材文件夹、脚本文件夹、输出数量 10 条、最大时长 35 秒，要求使用 `qianchuan-video` Skill 批量生成。
