# 项目 Skill 卡片：千川视频自动剪辑

【保存类型】
项目 Skill 卡片

【结论】
`qianchuan-video` 是可复用的千川产品视频自动剪辑流程：基于产品素材 + Excel 脚本，批量生成竖屏广告视频，自动完成画面匹配、TTS 配音、字幕、花字和背景音乐。

【步骤】
1. 准备素材目录：产品实拍、内料质地、模特效果、背景音乐。
2. 准备 Excel 脚本，包含动作、画面、语言/口播/文案字段。
3. 配置火山/豆包 TTS 密钥到本地 `secrets.local.json`，不要写入经验库或 Skill 包。
4. 调用 `qianchuan-video` Skill。
5. 批量生成视频，默认输出 1080x1920、30fps、AAC 音轨。
6. 配音按脚本文案逐句生成，字幕和画面跟随配音时间轴。
7. 背景音乐按素材池轮换，并随机起点。
8. 用 `ffprobe` 检查分辨率、帧率、音轨和时长。

【提示词】
使用 `qianchuan-video` Skill，根据这个素材文件夹和脚本文件夹，批量生成 10 条千川竖屏视频。要求：配音、字幕和画面逐句同步；视频时长跟随配音但不超过 35 秒；每条使用不同背景音乐；去除原声；字幕不要出现反斜杠；禁止静止补帧，画面要流畅。

【坑点】
- 强行压成 15 秒会导致配音过快。
- 用 `tpad=stop_mode=clone` 会造成画面假卡顿，应循环动态素材或拆镜头。
- ASS 字幕里的 `\N` 如果二次转义，会在画面显示反斜杠。
- 停止批处理时，要检查 Python/FFmpeg 子进程是否仍在跑。
- TTS API Key 不能写进 MD、模板、日志或 Skill 包。
- 长脚本不能硬塞，应按语义截取或限制最大时长。
- 换设备复用时，MD 只能说明流程；真正可执行需要 Skill、脚本、FFmpeg、Python、素材和本地密钥。
