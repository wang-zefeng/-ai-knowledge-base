## 3. 关键文件说明

| 文件 | 作用 |
| --- | --- |
| `src\App.tsx` | 主要前端状态、视图切换、Agent 工作室交互、表格分析入口。 |
| `src\services\ai-service.ts` | 统一 AI 调用层，当前通过 `generateAI()` 调用模型。 |
| `server.ts` | Express 服务端入口，包含 API、生产静态资源服务和部分业务路由。 |
| `package.json` | 项目脚本入口，重点看 `lint`、`build`、`start`、`dev`。 |
| `data\` | 当前 JSON 持久化数据目录，生产化前应迁移数据库。 |
| `dist\` | 构建产物，不进入经验库，不作为沉淀内容复制。 |
